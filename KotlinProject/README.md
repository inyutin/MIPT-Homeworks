# KotlinProject
Библиотека с реализацией основных методов поиска выпуклой оболочки.
Реализована в рамках курса "Kotlin на Физтехе".

## Зависимости
  - jdk 8
  - Kotlin
  - Maven
  
## Использование
  - Для сборки: `mvn install`
  - Для тестирования: `mvn test`
